# Netflix Landing Page Clone

A front-end clone of the Netflix landing page built with HTML and CSS for educational purposes.

> **Note:** This is a non-functional UI clone created for learning. Not affiliated with Netflix Inc.

## Features

- Responsive design for all screen sizes
- Pure HTML & CSS (no JavaScript frameworks)
- Accurate visual replication of Netflix UI

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## Getting Started

```bash
git clone https://github.com/TirthDadhaniya/netflix-clone.git
cd netflix-clone
# Open index.html in your browser
```

## Project Structure

```
netflix-clone/
├── index.html      # Main HTML file
├── style.css       # Stylesheet with responsive design
├── res/            # Images and videos
└── favicon.ico     # Website favicon
```
