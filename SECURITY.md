# Security Policy

## ⚠️ Important Notice

**This is an EDUCATIONAL PROJECT** - a static HTML/CSS demonstration that does NOT collect, process, or store any user data.

## Project Nature

This repository contains a **non-functional UI clone** of Netflix's landing page, created purely for:

- Learning HTML and CSS
- Portfolio demonstration
- Educational purposes

## Security Measures Implemented

1. **No Backend**: This project has no server-side code, database, or API connections
2. **No Data Collection**: All form inputs are disabled and non-functional
3. **No Authentication**: There is no login system or user account functionality
4. **No External Requests**: The site does not send data to any external servers
5. **Static Content Only**: Pure HTML and CSS with no dynamic functionality

## What This Project Does NOT Do

- ❌ Collect email addresses or any personal information
- ❌ Process form submissions
- ❌ Store cookies for tracking purposes
- ❌ Authenticate users
- ❌ Connect to Netflix or any third-party services
- ❌ Impersonate Netflix for malicious purposes
